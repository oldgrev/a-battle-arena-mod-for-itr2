# what? why???

muscle memory. I was going back to the game every ~6 months when there's been a big early access update and then starting again to "see all the content", but basically going over the same routine and basic early game weapons and scenarios.

I hoard, I make sure I have a lot of ammo and a lot of healing items before I go in. I reduce the risks as much as possible and the preparation changes the gameplay loop from a typical 33% prep, 33% exploration, 33% combat balance, to a 60% prep, 20% exploration, 20% combat balance based on time. 

So I end up fumbling reloads, and lack familiarity with the weapons, but sometimes i just want to have a bit of fun with the combat ai instead of the game's short ambushes every so often. i want to get better at accuracy and knowing how many shots are left, and knowing weapon accuracy and enemy behaviours etc, which I think answers the 'why?'.
